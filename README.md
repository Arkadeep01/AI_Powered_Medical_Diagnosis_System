## ⚙️ Core Components

### 🧠 Machine Learning Layer
- Trained on medical datasets  
- Supports:
  - Diabetes  
  - Heart Disease  
  - Lung Disease  
  - Parkinson’s  
  - Hypothyroid  
  - Migraine  
  - Fetal Health  
- Export formats:
  - `.sav`, `.pkl`, `.joblib`

---

### ⚙️ Backend (Flask)
- Loads ML models at startup  
- Provides API endpoints (`/predict`, `/analyze`)  
- Handles preprocessing and inference  
- Returns structured JSON responses  

---

### 🎨 Frontend (React Dashboard)
- Interactive disease selection  
- Real-time form inputs  
- Normal range guidance display  
- Clean dark-themed UI  
- Dynamic prediction results  

---

### 📦 Deployment (Docker)
- Fully containerized system  
- Includes backend, frontend build, and ML models  
- Ensures portability and consistency
